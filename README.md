# Ethics Personal Statement
## My Views on Ethics

One of the things I dislike most is when people are not completley honest with me or misleading. That is why when working with and building AI, it is vital to ensure I am completely transparent with my users. 
As people give AI more and more data, we should begin being transparent about potential bias in the model and accuracy of the AI. This becomes more important since we are being forced to use AI more everyday.
When making an AI model, I will research the data used to make the model and report on any potential bias. If any are found, the user will be made aware. Thorough testing will also be conducted to see if human oversight makes more accurate decisions. The decision based on these tests will be disclosed to the user as well. Furthermore, I will keep monitoring the AI model with user feedback and report on any flaws done when it reaches a wide userbase.
